# 使い方

- クエリパラメータ

```
?idol=35&event=142&rank=1,10,100,1000
```

|key | 内容|
|----| ---|
| idol | アイドルのID。1～52のどれか。
|event | イベントのID。3周年なら142。
|rank  | 取得対象の順位をカンマ区切りで。<br>指定する数は5個以内。<br>1000より大きい順位はデータが無い。

- データは30分ごとに更新。データの取得元が集計中だったりすると画面反映が少し遅れることがある。
